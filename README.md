# dijkstra
A python script to implement and visualize dijkstra algorithm.


## Instructions to run

* Download the python script and run it in your favourite editor or use the command
```
python3 dijkstra_charith_reddy.py
```

## Libraries Used:
1. pygame
2. math
3. heapq
4. time
